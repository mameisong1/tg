## Bug 修复记录

### BUG-0415-01: UNION ALL ORDER BY 类型不匹配

- **发现**: Preview API 返回 500 错误
- **错误信息**: `SQLITE_ERROR: 1st ORDER BY term does not match any column in the result set`
- **根因**: `coaches.coach_no` 是 INTEGER，`water_boards.coach_no` 是 TEXT，SQLite 的 UNION ALL 中 `ORDER BY CAST(...)` 不支持直接对 UNION 结果用表达式排序
- **修复**: 用子查询包裹 UNION ALL，在外部查询中 ORDER BY CAST
  ```sql
  SELECT * FROM (
    SELECT ... UNION ALL SELECT ...
  ) ORDER BY CAST(coach_no AS INTEGER)
  ```
- **额外修复**: execute API 中 `SELECT FROM coaches WHERE coach_no = ?` 也需要 CAST 匹配
- **状态**: ✅ 已修复，3 次 commit
