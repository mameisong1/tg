# QA1 测试用例 — 助教乐捐报备流程修改

> 测试日期：2026-04-15
> 测试员：B1
> 测试环境：后端 http://127.0.0.1:8088 | 前端 http://127.0.0.1:8089

---

## 功能概述

助教乐捐报备流程：
1. **预约整点开始时间** → 助教提交乐捐预约，选择整点时间
2. **到时间自动变乐捐状态** → 定时器到时间后自动将水牌状态改为「乐捐」
3. **乐捐归来** → 助教管理在后台点击「乐捐归来」，计算外出小时数并恢复水牌为空闲
4. **近2天记录可提交付款截图** → 助教可在近2天的记录中上传/修改付款截图

## 状态流转图

```
助教提交预约
    ↓
[pending] 待出发 → 定时器触发 → [active] 乐捐中
    ↓                              ↓
    └── 助教管理「乐捐归来」 ←──────┘
    ↓
[returned] 已归来（记录外出小时数，恢复水牌空闲）
```

## API 接口清单

| 接口 | 方法 | 路径 | 权限 | 说明 |
|------|------|------|------|------|
| 提交预约 | POST | /api/lejuan-records | 助教 | 创建 pending 状态的乐捐记录 |
| 我的记录 | GET | /api/lejuan-records/my | 助教 | 查询近2天记录 |
| 乐捐一览 | GET | /api/lejuan-records/list | 助教管理 | 查看所有记录 |
| 乐捐归来 | POST | /api/lejuan-records/:id/return | 助教管理 | 计算外出小时，恢复空闲 |
| 提交截图 | PUT | /api/lejuan-records/:id/proof | 助教 | 上传/修改付款截图 |
