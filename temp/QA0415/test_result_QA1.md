## Bug 说明

### Bug 1: TC-13 失败 — 测试数据问题（非功能 Bug）

**现象：** TC-13 需要找到 pending 和 returned 状态的记录来测试「非 active 状态乐捐归来应拒绝」，但测试流程中唯一的 pending 记录被 TC-03 手动激活，又被 TC-04 变为 returned 后删除了。

**根因：** 测试用例执行顺序导致前置数据被后续用例消耗。

**建议：** 为 TC-13 单独创建测试数据，不依赖其他用例的副作用。

### Bug 2: TC-07 失败 — UI 选择器不匹配（脚本 Bug）

**现象：** Puppeteer 找不到 `input[name="username"]` 选择器。

**根因：** 登录页使用的是 `id="username"` 而非 `name="username"`。

**修复方案：** 将选择器改为 `#username` 和 `#password`。
