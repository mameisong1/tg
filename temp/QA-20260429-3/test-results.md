## 发现的问题

### 1. logCron函数未导出（已修复）

**问题**：`cron-scheduler.js` 中 `logCron` 函数未在 `module.exports` 中导出，导致 `server.js` 调用时报错 `logCron is not a function`

**影响**：TC-01首次执行返回服务器错误

**修复**：在 `cron-scheduler.js` 的 `module.exports` 中添加 `logCron`

**修复代码**：
```javascript
module.exports = {
    init,
    triggerTask,
    getTaskStatus,
    getAllTasks,
    getCronLogs,
    toggleTask,
    calcNextRun,
    logCron,  // 新增
    // ...
};
```

**状态**：已修复并重启服务验证
