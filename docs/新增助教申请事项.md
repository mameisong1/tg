#### QA2. 申请表新增申请事项：切换班次申请，请假申请，休息申请。
applications.application_type 申请类型修改：新增（班次切换申请/请假申请/休息申请），删除（乐捐报备）
前台H5会员中心:
- 助教专用版块新增3个页面：班次切换申请，请假申请，休息申请。权限：助教权限和后台用户权限都可用。
- 管理功能版块新增3个页面：班次切换审批，请假审批，休息审批。权限：助教管理/店长/管理员。

1. 班次切换
- 助教提交班次切换申请，助教管理或店长审批同意后，该助教的班次自动切换早晚班；
- 申请页面显示：当前班次，本月已申请（审批通过）次数。
- 限制：每人每月2次，如果本月已经审批同意过2次切换申请，再次提交时报错，告知原因。
- 审批页面显示：当前早班和晚班助教人数。审批同意后修改助教水牌里的班次: 如果是空闲/上桌/加班状态要切换早晚班。

2. 助教休息申请
- 助教一个月内可以申请4天休息日（不需要任何理由）。
- 一次申请一天，可选当天或未来30天内的某天。
- 当月已经申请过4天，提交时报错告知理由，不让提交。
- 申请页面底部可以看到近30天内提交的休息申请记录。已经提交还未审批的，助教自己可以取消（直接删除记录）。
- 审批页面显示：当日预计休息+请假人数。
- 审批同意后水牌状态要变更：设定执行计时器预约到申请的休息日12点执行：自动从（下班/公休/请假/乐捐）等状态变休息状态。(如果审批时间>休息日12点，那么立刻执行)
- 审批同意后，在到达申请的休息日12点之前，助教可以找助教管理要求取消申请（把记录删除，同时关闭执行计时器）

3. 助教请假申请
- 请假分事假和病假，页面提供事假/病假选择，必须输入请假理由，可选提交最多3张照片（照片提交功能使用公共模块，参考公休申请页面）。
- 一次申请一天，请假日期可选当天或未来30天内的某天。
- 申请页面底部可以看到近30天内提交的请假申请记录。已经提交还未审批的，助教自己可以取消（直接删除记录）。可以在已提交的请假记录里面补充提交照片。
- 审批页面显示：当日预计休息+请假人数， 事假/病假，理由，照片。
- 审批同意后水牌状态要变更：设定执行计时器预约到申请的请假日12点执行：自动从（下班/公休/请假/乐捐）等状态变请假状态。(如果审批时间>请假日12点，那么立刻执行)
- 审批同意后，在到达申请的请假日12点之前，助教可以找助教管理要求取消申请（把记录删除，同时关闭执行计时器）

4. 待审批事项指示数字显示
- 新增api，用一个sql查询当前所有待审批的事项数量：加班（早班+晚班）申请数， 公休申请数， 班次切换申请数， 休息申请数，请假申请数，约客待审查人数。
- 助教管理/店长/管理员进入前台H5会员中心页面时，调用api查询数量，然后在管理功能相应按钮的右上角显示待审批的指示数字（类似app图标右上角的数字）
