### TC-008: 批量更新班次 - 空选择提交 ❌ 失败

| 项目 | 状态 | 说明 |
|------|------|------|
| **执行状态** | ❌ **失败** | **实现方式与用例描述不符** |
| **实际实现** | 批量更新班次页面采用**逐个修改**模式，每位助教有自己的「早班/晚班」切换按钮 | **没有"全选/多选"功能**，也没有"选择2名助教后提交"的流程 |
| **实际流程** | 为每位助教选择班次 → 点击「全部保存」→ 所有修改生效 | 与测试用例描述的全选→选择班次→提交流程不一致 |
| **页面标题** | "为每位助教选择班次，修改后点击「全部保存」生效" | 设计逻辑是批量逐个设置，而非批量选择 |
| **建议** | 测试用例需更新以匹配实际实现，或与产品经理确认是否需改为全选模式 |
