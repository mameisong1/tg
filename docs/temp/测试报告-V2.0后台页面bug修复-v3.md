**🔴 批量更新班次模块存在两个关键Bug：**

### BUG-001: 数据加载时序问题
- **现象**: 直接访问 `coaches.html#batch-shift` 时显示「暂无助教数据」
- **根因**: `checkHash()` 在页面加载时同步调用 `renderBatchShift()`，但 `loadCoaches()` 的API请求尚未完成，`coaches` 数组为空
- **修复建议**: 在 `loadCoaches()` 的 `renderCoaches()` 之后，追加判断：如果当前hash为 `#batch-shift`，调用 `renderBatchShift()`

### BUG-002: 班次选择器未使用单选按钮
- **现象**: 班次选择器使用 `<select>` 下拉框
- **需求**: TC-403/TC-404明确要求「使用单选按钮（Radio Button），不得使用下拉框（Select）」
- **修复建议**: 将 `<select>` 替换为 radio button，如：
  ```html
  <label><input type="radio" name="shift-${coach_no}" value="早班" ${checked}> 早班</label>
  <label><input type="radio" name="shift-${coach_no}" value="晚班" ${checked}> 晚班</label>
  ```

**本模块小结**: 8条用例中 3条通过，1条部分通过，**4条失败**。
