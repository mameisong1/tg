## 水牌状态测试用例

### TC-006: 水牌状态完整性验证
- 前置条件：
  - 已登录具有水牌管理权限的账号
  - 水牌管理页面已打开
- 测试步骤：
  1. 进入水牌管理页面
  2. 查看状态选择下拉框
  3. 统计所有可选状态数量
- 预期结果：
  - 系统应包含全部 11 种状态：
    1. 早班上桌
    2. 早班空闲
    3. 晚班上桌
    4. 晚班空闲
    5. 早加班
    6. 晚加班
    7. 休息
    8. 公休
    9. 请假
    10. 乐捐
    11. 下班

### TC-007: 水牌状态切换验证
- 前置条件：
  - 已登录具有水牌管理权限的账号
  - 存在至少一个教练水牌
- 测试步骤：
  1. 选择任意教练水牌
  2. 将状态从"下班"切换为"早班上桌"
  3. 保存并刷新页面
  4. 再次切换状态为"晚班空闲"
  5. 保存并刷新页面
- 预期结果：
  - 状态切换成功
  - 刷新后状态保持最新选择
  - 状态变更有记录日志

## 约客管理测试用例

### TC-017: 早班约客上传截止时间验证
- 前置条件：
  - 已登录具有约客管理权限的账号
  - 当前时间在 16:00 之前
- 测试步骤：
  1. 在 15:59 尝试上传早班约客
  2. 在 16:00 尝试上传早班约客
  3. 在 16:01 尝试上传早班约客
- 预期结果：
  - 15:59 可以成功上传早班约客
  - 16:00 可以成功上传早班约客（含截止时间点）
  - 16:01 上传失败，提示"早班约客上传已截止"

### TC-018: 晚班约客上传截止时间验证
- 前置条件：
  - 已登录具有约客管理权限的账号
  - 当前时间在 20:00 之前
- 测试步骤：
  1. 在 19:59 尝试上传晚班约客
  2. 在 20:00 尝试上传晚班约客
  3. 在 20:01 尝试上传晚班约客
- 预期结果：
  - 19:59 可以成功上传晚班约客
  - 20:00 可以成功上传晚班约客（含截止时间点）
  - 20:01 上传失败，提示"晚班约客上传已截止"

### TC-019: 应约客列表筛选验证
- 前置条件：
  - 已登录具有约客管理权限的账号
  - 存在多个不同状态的教练
- 测试步骤：
  1. 进入约客管理页面
  2. 查看"应约客"列表
  3. 检查列表中教练的水牌状态
- 预期结果：
  - "应约客"列表仅包含状态为"早班空闲"或"晚班空闲"的教练
  - 状态为"早班上桌"、"晚班上桌"、"休息"、"请假"等的教练不在应约客列表中

### TC-020: 非空闲状态不应约验证
- 前置条件：
  - 已登录具有约客管理权限的账号
  - 存在状态为"早班上桌"的教练
- 测试步骤：
  1. 进入约客管理页面
  2. 在"应约客"列表中查找状态为"早班上桌"的教练
- 预期结果：
  - "应约客"列表中不包含状态为"早班上桌"的教练
  - 该教练可能出现在"已上桌"或其他分类中

### TC-021: 约客上传内容验证
- 前置条件：
  - 已登录具有约客管理权限的账号
  - 在截止时间前
- 测试步骤：
  1. 选择早班约客上传
  2. 填写约客信息（客户姓名、联系方式、预约时间等）
  3. 提交保存
  4. 刷新页面查看
- 预期结果：
  - 约客信息成功保存
  - 刷新后数据不丢失
  - 约客状态显示为"待应约"

## 综合场景测试用例

### TC-022: 完整班次流程验证
- 前置条件：
  - 已登录具有水牌管理权限的账号
  - 教练状态为"下班"
- 测试步骤：
  1. 教练点击"上班"，选择早班
  2. 状态变为"早班空闲"
  3. 教练"上桌"，状态变为"早班上桌"
  4. 教练"下桌"，状态变为"早班空闲"
  5. 教练点击"下班"
  6. 状态变为"下班"
- 预期结果：
  - 所有状态转换正确
  - 每个操作都有时间记录
  - 班次统计准确

### TC-023: 权限与水牌组合验证
- 前置条件：
  - 已创建教练角色账号
  - 已创建店长角色账号
- 测试步骤：
  1. 使用教练账号登录，查看水牌页面
  2. 尝试修改水牌状态
  3. 使用店长账号登录，查看水牌页面
  4. 修改水牌状态
- 预期结果：
  - 教练只能查看水牌，无法修改状态
  - 店长可以查看并修改水牌状态

### TC-024: 约客截止后状态验证
- 前置条件：
  - 已登录具有约客管理权限的账号
  - 当前时间已超过 16:00
- 测试步骤：
  1. 尝试上传早班约客
  2. 查看系统提示
  3. 尝试上传晚班约客（时间未到 20:00）
- 预期结果：
  - 早班约客上传被拒绝，提示已截止
  - 晚班约客可以正常上传
