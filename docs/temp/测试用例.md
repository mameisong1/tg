## 一、状态定义

| 状态分类 | 具体状态 |
|----------|----------|
| 上桌状态 | 早班上桌、晚班上桌 |
| 空闲状态 | 早班空闲、晚班空闲 |
| 乐捐状态 | 乐捐 |
| 加班状态 | 早加班、晚加班 |
| 不在店状态 | 休息、公休、请假、下班 |
