## 一、助教管理页面 — 班次字段 CRUD 操作

### TC-001: 助教列表中显示班次字段
| 项目 | 内容 |
|------|------|
| 前置条件 | 已登录后台，数据库中存在至少1名助教，助教表中已有班次字段（早班/晚班） |
| 操作步骤 | 1. 访问 `/admin/coaches.html`<br>2. 查看助教列表表格/卡片 |
| 预期结果 | 每条助教记录中显示「班次」字段，值为「早班」或「晚班」之一 |

### TC-002: 新增助教时选择班次
| 项目 | 内容 |
|------|------|
| 前置条件 | 已登录后台，进入助教管理页面 |
| 操作步骤 | 1. 点击「新增助教」按钮<br>2. 在新增表单中找到班次字段<br>3. 选择「晚班」<br>4. 填写其他必填信息，点击保存 |
| 预期结果 | - 班次下拉选项仅有「早班」和「晚班」两个选项（不应有「全天」「休息」等）<br>- 保存成功，新助教的班次为「晚班」<br>- 列表中新增的助教显示对应班次 |

### TC-003: 编辑助教时修改班次
| 项目 | 内容 |
|------|------|
| 前置条件 | 数据库中有助教A，班次为「早班」 |
| 操作步骤 | 1. 在助教列表中找到助教A，点击编辑<br>2. 将班次从「早班」改为「晚班」<br>3. 点击保存 |
| 预期结果 | - 保存成功<br>- 助教列表中助教A的班次显示为「晚班」<br>- 数据库中该助教 shift 字段已更新为「晚班」 |

### TC-004: 删除助教
| 项目 | 内容 |
|------|------|
| 前置条件 | 数据库中存在可删除的助教 |

| 预期结果 | - 弹出确认对话框<br>- 确认后删除成功，该助教从列表中消失<br>- 数据库中该记录已删除或标记为已删除 |
