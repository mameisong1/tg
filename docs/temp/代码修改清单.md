## 数据库脚本

数据库迁移脚本已存在：`backend/db/migrations/v2.0.sql`，包含以下操作：

```sql
-- 1. admin_users 表 role 字段索引
CREATE INDEX IF NOT EXISTS idx_admin_users_role ON admin_users(role);

-- 2. coaches 表新增班次字段
ALTER TABLE coaches ADD COLUMN shift TEXT DEFAULT '早班';

-- 3. 创建新表
-- water_boards - 水牌表
-- service_orders - 服务单表
-- table_action_orders - 上下桌单表
-- applications - 申请事项表
-- guest_invitation_results - 约客结果表
-- operation_logs - 操作日志表

-- 4. 初始化水牌数据
INSERT OR IGNORE INTO water_boards (coach_no, stage_name, status, table_no)
SELECT coach_no, stage_name, '下班', NULL FROM coaches;
```

**注意**: 数据库迁移脚本未做修改，已满足需求。
