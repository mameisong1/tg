# 水牌状态变化测试用例

## 测试用例概述

本测试用例覆盖水牌状态变化的两个维度：
1. **允许/禁止验证**：验证各操作在不同当前状态下是否允许执行
2. **状态变化验证**：验证操作执行后水牌状态的正确变化

## 测试注意事项

1. **班次验证**：所有状态变化需验证早班/晚班的正确性，根据助教表.班次字段决定
2. **状态一致性**：上桌/空闲状态必须保持班次一致性（早班上桌→早班空闲）
3. **禁止操作提示**：所有禁止操作需验证提示信息的正确性
4. **审批状态**：申请类操作（加班晚到、公休）不影响当前状态，审批后才变化
5. **乐捐特殊性**：乐捐报备可使任何状态变为乐捐，包括下班状态
